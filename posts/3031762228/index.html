前言

基于第一章的准备工作，现在开始搭建+部署博客

本地搭建

安装hexo

在桌面右键，在弹出来的窗口中单击“Git Bush Here”

在弹出来的窗口中输入要创建博客的地址
比如我的要搭建在F盘
那么我就输入

cd /f/


如图，成功跳到了F盘

然后在git窗口中输入

npm install hexo-cli -g

结果如图（差不多即可）

再输入

hexo init myblog

速度有点慢，需要稍等一会
结果如图（差不多即可）（以下省略）

同时，你的（D，E，F……）盘下会出现应该叫“myblog”的文件夹
欧克，用git打开myblog这个文件夹

cd myblog
